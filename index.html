What is Ren Bridge?

RenBridge is the Ren project’s bridge UI for RenVM, a system historically used to mint and burn representations such as renBTC across chains. The Ren Resource Center documents RenVM, RenJS, Darknodes, and developer resources.

How Ren Bridge works

  1. Open the appUse the official domain and confirm the URL before connecting any wallet.
  2. Choose a routeSelect the asset, source network, and destination network shown as available in the interface.
  3. Review detailsCheck destination address, wallet network, confirmations, gas, and any route-specific notices.
  4. Sign and monitorApprove only the expected transaction, then track source and destination confirmations.

Before you bridge

Official route first

The public bridge-v2 repository identifies bridge.renproject.io as the RenBridge app domain.

Built around RenVM

RenVM was designed for cross-chain asset movement, including mint and burn flows for wrapped representations such as renBTC.

Verify current support

Routes, assets, and availability can change, so rely on the live app state and official project resources instead of old screenshots or third-party mirrors.

Ren Bridge FAQ

What is ren bridge?

ren bridge is commonly used to refer to RenBridge, the Ren project’s app interface for RenVM cross-chain asset flows.

Is ren bridge the official app?

The official app domain referenced by the RenBridge source repository is bridge.renproject.io; verify the domain before connecting.

What can I move with ren bridge?

Historically, RenVM supported mint and burn routes for assets such as BTC into representations like renBTC, but current availability should be checked in the app.

Does ren bridge show fees before signing?

Use the app review screen to inspect route details, network gas, and transaction prompts before approving anything; this page does not quote live fees.

What if ren bridge does not load?

Do not use look-alike domains. Check official Ren documentation and the source repository for current project references.
